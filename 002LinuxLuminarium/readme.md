# LinuxLuminarium
Basic commands

- Man is stored in a separate indexed db which can be modified
- FD - {0: std input, 1: std output, 2: std error}
- Linux follows: "Everything is a file property."
- Linux ownership mechanism and how it includes authorization using ACL[Access Control Lists]. (quite interesting)
- ```tee``` allows us to copy data while piping
- ";" runs multiple bash commands
- "&&" runs new command only if previous command was succesful
- "||" runs new command only if previous command failed
